Gestor de Asistencia

Una aplicacion de escritorio para registrar asistencia diaria, consultar informes y trabajar con alumnos, asignaturas y grupos desde una interfaz clara. El instalador incluye su propio entorno Java, asi que la puesta en marcha es directa.

Sistema: Windows Java incluido Datos: MongoDB Atlas

Pensado para explicar el proyecto en dos minutos.

La pagina resume el valor principal de la app con una estetica reconocible, pero mantiene el foco en lo importante: tomar asistencia, revisar datos y descargar el entregable correcto sin pasos confusos.

Modulo 1
Asistencia diaria

Selecciona fecha, asignatura y grupo para registrar el estado de cada alumno.

Modulo 2
Informes

Calcula porcentajes de faltas y ayuda a localizar casos de seguimiento.

Modulo 3
Usuarios

Incluye acceso para administracion y profesorado con datos de prueba.

Modulo 4
Exportacion

La interfaz contempla salidas JSON y PDF para compartir informacion.
